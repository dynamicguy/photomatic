Installation
--------------

1. Clone the repo `git clone git://github.com/dynamicguy/photomatic.git`
2. `cd photomatic`
3. `mkvirtualenv venv`
4. `workon venv`
5. `pip install -r requirements.txt`
6. `python manage.py syncdb`
7. `python manage.py runserver`
